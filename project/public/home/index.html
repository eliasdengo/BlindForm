About Us

The general objective of this project is to develop Web based Chatting System that provides anonymous forum and community for user/Client.

  • Provides multiple discussion forums where user can connect with other user or client.
  • Provides a private chat (i.e the system provides one to one conversation by matching two interested clients on private chat) and group chat channel to discuss all things work-related with just your co-workers.
  • Provides free talking community through implicating users information.
  • Allows users to chat in their interest

Features

Freedom of speech

This system provides freedom for users to penetrate significant information freely to expected person. The system is more secured since it does not provide user information to chat in the system.

Anonymous

Generally the system provides chatting system for anonymous users. It provide applications that will make the processing of chatting very easy and secure

Security

The massage that is transferred between the users is secured. This app insure that TLS (transfer-level security) which is supported by the streaming technology that we will use which is RabbitMQ.
